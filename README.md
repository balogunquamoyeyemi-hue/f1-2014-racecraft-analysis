# Formula 1 2024: Racecraft, Dominance & Driver Performance Analysis 

**OVERVIEW**

     This Project presents an in-depth analysis of the 2014 Formula 1 season, focusing on driver racecraft, performance consistency, and competitive dominance.
     The dashboard goes beyond basic race result to highlight position gains, win efficiency, and reliability trends across drivers and teams.

**OBJECTIVES**
- Identify the most dominant drivers based on win rate and performance
- Analyze position gains to evaluate racecraft and overtaking ability
- Examine driver reliability through race retirements (DNFs)
- Compare driver efficiency using points earned per race
- Evaluate the relationship between starting grid and race outcomes

**KEY INSIGHTS**
- A small group of drivers dominated race wins and overall performances
- Drivers with strong position gains demonstrated superior racecraft
- Reliability grid position impacted several drivers performance
- Starting grid position strongly influenced final race outcomes
- Points efficiency revealed performance consistency beyong race wins

**TOOLS USED**
- Power BI
- Power Query (Data Cleaning)
- DAX (Measures & Calculations)
